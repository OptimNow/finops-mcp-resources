## 1. Overview

The Azure MCP server lets you query **FinOps Hub** and **Azure Cost Management** data directly in your MCP client (like VS Code). Use it for:

- Cost analysis
- Optimization insights
- FinOps workflow automation

------



## 2. Prerequisites

- An **Azure subscription** with a [FinOps Hub](https://learn.microsoft.com/en-us/cloud-computing/finops/toolkit/hubs/finops-hubs-overview?utm_source=chatgpt.com#create-a-new-hub) and **Data Explorer** enabled. *ðŸ‘‰ For setup details, see the [Azure FinOps Hub guide](https://learn.microsoft.com/en-us/cloud-computing/finops/toolkit/hubs/finops-hubs-overview?utm_source=chatgpt.com#create-a-new-hub).*
- **VS Code** installed and running. 
- Azure permissions: assign yourself or your service principal the **Cost Management Reader** role (and optionally **Reader** at resource group level for more detail)



------

## 4. Minimal Config

If you want to check it manually, add this to your `mcp.json`:

```
{
  "mcpServers": {
    "azure-mcp": {
      "command": "npx",
      "args": ["-y", "@azure/mcp@latest", "server", "start"],
      "env": {
        "AZURE_SUBSCRIPTION_ID": "<your-subscription-id>"
      }
    }
  }
}
```
